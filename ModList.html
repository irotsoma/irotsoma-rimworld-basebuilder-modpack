irotsoma Patches:
Industrialization Patch Fixes for shadow related errors and fixes for the copper ore coloring (I really didn't like that bright green).
Pawn State Icons: Updated dll to work with alpha 12d
Roof Support: Updated xml file to work with alpha 12d
CTS: Updated xml files to work with alpha 12d
Remove Processed Resources This patch removes the refined steel bar and plasteel bar as well as the wood plank from Superior Crafting and any mods that depend on it. WARNING: This will overwrite files in the affected mods and can not be undone.
Category Patches This is a collection of mods that alter the categorization of objects, the Architect menu, and research. It follows the research pattern originally set out by Superior Crafting, thus Superior Crafting is a prerequisite.
Incompatibilities Patches This is a collection of mods that remove duplicate or redundant features and patches some incompatibilities between mods. Only activate the mod if you are using all of the mods mentioned, and make sure to put these patches after, in the mod order, all of the mods mentioned for the particular patch.
